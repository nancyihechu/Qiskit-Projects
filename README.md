# Qiskit Quantum Projects

Welcome to my repository showcasing various quantum computing experiments and projects using **Qiskit**. In this repository, you'll find a combination of **Jupyter Notebooks**, **PDFs** of quantum math assignments, and **Quantum Composer** solutions. The primary focus of these projects is on quantum circuits, quantum algorithms, and running simulations on real quantum devices using IBM's quantum computing platform.

## Getting Started

To run the **Jupyter Notebook** and experiment with quantum circuits:
1. Install **Qiskit** by following the instructions at [Qiskit Install Guide](https://qiskit.org/documentation/getting_started.html).
2. Open the **Introduction to Qiskit** notebook in Jupyter, and run through the examples provided. You can modify the circuits and try your own quantum operations.
3. The PDFs are available for reference; feel free to review the solutions to deepen your understanding of quantum math and quantum computing concepts.
